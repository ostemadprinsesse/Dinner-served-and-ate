##Dagens opgaver

Go through this checklist before the weekend starts:

 Make sure to have a group organization on github
 Make sure to have a cookbook repository, with your own framework code
 Make sure to have a the groups.py file up to date with repo url, group memebers and stack information.
 Make sure to have OpenApi REST api documentation added to your framework code
 Make sure that when you think your code is as it should be (for now), to merged it into the master branch.
 Create tags and releases
 Add a dockerfile and a docker-compose file to your repository (it helps everybody to run you application)
 Write a simple description in the repos readmefile on how to run your application
 Make sure to Have set up a project and a Kanban board
 Check you issues tab for potetially issues added by me, or others
